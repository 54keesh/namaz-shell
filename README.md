# Next Prayer

Showing the time for the next Islamic prayer.

---

## Installation

```
git clone https://github.com/AbdeltwabMF/Next-Prayer.git
cd Next-Prayer
./install
NextPrayer
```

---

## Dependencies
  - `jq`
  - `python3`
  - `gawk`
